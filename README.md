# cekIP
Hai,
Script ini digunakan  untuk mengecek Indek Prestasi(IP) Mahasiswa STIKI INDONESIA.


Cara Penggunaan:

./cekIP.sh [masukkan NIM mahasiswa yang ingin dicari]


Contoh :

./cekIP.sh 16101567
